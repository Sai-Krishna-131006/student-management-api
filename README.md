# Spring Boot Backend Demo

This project demonstrates a basic REST API built using Spring Boot.

## Features
- GET API to retrieve student data
- POST API to create student data
- JSON request and response handling
- Tested using Postman

## Technologies
- Java
- Spring Boot
- Maven
- Postman

## API Endpoints

GET /student  
GET /students  
POST /student
